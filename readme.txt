# This is a README file to compile the coupled model si3d-gotm

- Copy gotm directory in your files
 
- 

1) save'Makefile' and 'compile' in the si3d source code folder

+++ Compile gotm with IFORT

2) specify the path of the gotm folder in the 'Makefile'
   (substitute '/home/agua/gotm-cvs/src' by the path of your system)

3)  specify the path of the gotm and si3d folder in the 'compile' file
   (substitute '/home/usr/gotm-cvs/src' and '/home/usr/si3d/src' by the path of your system)

4) Run the compile file (./compile)
